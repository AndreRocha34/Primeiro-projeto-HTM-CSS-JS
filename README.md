## **README.md - Sua Horta em Casa: Um Guia Simples e Rápido**

###  **O que é essa aplicação?**

Essa aplicação web te ajuda a descobrir tudo o que você precisa saber para cultivar sua própria horta em casa! 

**Como funciona:**

1. **Pesquise:** Digite o nome da verdura ou legume que você quer cultivar no campo de pesquisa.
2. **Descubra:** A aplicação te mostra um guia rápido com as informações mais importantes, como:
    * **Onde encontrar mais informações:** Links para artigos completos sobre o cultivo.
    * **Dicas rápidas:** Informações básicas para começar a cultivar.

###  **Tecnologias utilizadas:**

* **HTML:** A estrutura básica da página.
* **CSS:** Para deixar tudo bonitinho e organizado.
* **JavaScript:** A mágica por trás da pesquisa e da exibição dos resultados.

###  **Como usar:**

1. **Clone o repositório:** Use o Git para clonar esse projeto para sua máquina.
2. **Abra o arquivo index.html:** Use seu navegador favorito para abrir o arquivo index.html.
3. **Comece a pesquisar:** Digite o nome da planta que você quer cultivar e veja a magia acontecer!
